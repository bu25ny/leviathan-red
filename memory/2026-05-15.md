## Experience Reflection

### Lecciones de ingeniería

**1. Never use heredoc with Python `from` statements.**
El patrón `python3 << 'EOF' ... from datetime import ...` hace que el shell interprete `from` como un comando. Solución definitiva: escribir scripts Python a archivo con `write_file` → ejecutar con `python3 script.py`.

**2. Python 3.10 f-strings + backslashes.**
`\n` dentro de `{expresion}` en una f-string causa `SyntaxError` en Python 3.10. Nunca poner backslashes dentro de expresiones f-string. Extraer a variable temporal antes.

**3. PID mismatch en procesos padre/hijo.**
No calcular paths de archivos basados en `os.getpid()` cuando el hijo fork'd necesita referenciarlos. Pasar rutas como argumentos de línea de comandos (`sys.argv[1:]`).

**4. Cross-device link en `os.replace()` no existe.**
`tempfile.NamedTemporaryFile` sin `dir=` puede crear el archivo en un filesystem diferente al destino. Siempre especificar `dir=target_parent` para garantizar el `os.replace()` atómico funcione.

**5. El liveness probe debe testear el código real.**
Un probe genérico (script fijo de sleep) no detecta lógica catastrófica en el código candidato. Debe usar `importlib.util.spec_from_file_location()` + `exec_module()` para importar y ejecutar el código real, forzando que cualquier error de runtime se manifieste dentro del timeout.

**6. Pipeline A/B como gatekeeper real.**
El pipeline de despliegue existe para proteger v_stable. Solo `deployment_manager.deploy()` toca el sistema de archivos de producción. Ninguna escritura directa. Esta disciplina se prueba en Órden 03.
