## 🎓 Reflexiones y lógica reusable

### Sobre el Pipeline A/B
- **Regla inmutable:** Nada toca v_stable sin pasar por deployment_manager.deploy()
- **Liveness probe debe importar el módulo real** — un probe genérico no detecta errores en el código candidato
- **PID mismatch** entre padre e hijo es un bug clásico cuando se usa `os.getpid()` para nombrar archivos temporales en probes
- **Fail-safe en todos los niveles:** incluso si la DB no existe, el sistema debe seguir operativo

### Sobre escritura de scripts Python en este entorno
- **Heredocs (`<< 'EOF'`) fallan** en el shell sh de este sistema — siempre usar scripts escritos a archivo
- **`ast.parse()` como gatekeeper** — previene errores de sintaxis antes de I/O, reduce el ciclo de debug
- **f-strings con `\n` dentro de `{}`** son inválidos en Python 3.10 — extraer a variables externas
- **Path.flush()** no existe — siempre usar `with open()` para obtener file handles

### Sobre credenciales
- Copaw inyecta `OPENROUTER_API_KEY` como variable de shell (no necesita `.env`)
- `.env` tiene placeholders; credenciales reales viven en el entorno del framework
